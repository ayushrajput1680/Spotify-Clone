A simple and responsive Spotify Clone built using HTML, CSS, and JavaScript.
This project replicates the basic look and feel of Spotify’s music player interface. Users can play, pause, change songs, and navigate through the playlist with an interactive UI.

** Features**

Modern and responsive UI

Play, pause, next, previous controls

Progress bar & song timer

Playlist section

Smooth animations

Pure HTML, CSS & JavaScript — no frameworks

** Tech Stack**

HTML5 for structure

CSS for styling

JS for functionality

 **Purpose**

This project was created to practice frontend development, DOM manipulation, and working with music controls using JS.
